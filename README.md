# HTML to TSX & Tailwind CSS Standalone Tool

เครื่องมือแปลงโค้ด HTML และ CSS แบบดั้งเดิม (Legacy Code) ให้กลายเป็นโครงสร้าง React TypeScript (TSX) Component และจัดระเบียบสไตล์ด้วย Tailwind CSS Utility Classes แบบอัจฉริยะผ่านขุมพลังของ Gemini API โดยทำงานบนฝั่งผู้ใช้งานทั้งหมด (100% Client-Side) โดยไม่ต้องติดตั้งโปรแกรมใด ๆ

พัฒนาและสร้างสรรค์โดย Neon13

## 🌟 ฟีเจอร์เด่น (Key Features)

AI-Powered Conversion: ใช้โมเดล Gemini 2.5 Flash ในการวิเคราะห์โครงสร้าง HTML และแปลงสไตล์ดั้งเดิม (ทั้งแท็ก <style> และสไตล์ Inline) ไปเป็นคลาส Tailwind CSS ที่ใกล้เคียงที่สุด

Single-File Architecture: ทำงานบนไฟล์ index.html เพียงไฟล์เดียว ดับเบิลคลิกเปิดบนเว็บเบราว์เซอร์ใช้งานได้ทันที ไม่ต้องผ่านกระบวนการ Build หรือตั้งค่าระบบหลังบ้านให้ยุ่งยาก

Fully Responsive Design: ส่วนต่อประสานผู้ใช้ (UI) ออกแบบด้วยสไตล์โมเดิร์น (Dark Mode) และรองรับการแสดงผลสวยงามในทุกอุปกรณ์ (มือถือ, แท็บเล็ต, เดสก์ท็อป)

Built-in Presets: มาพร้อมโค้ดตัวอย่างในตัวระบบ (การ์ดโปรไฟล์ และ แบบฟอร์มล็อกอิน) เพื่อทดลองใช้งานและทดสอบระบบได้ทันที

Key Security & Convenience: รองรับการบันทึก API Key บนเบราว์เซอร์ผ่าน localStorage โดยไม่มีการส่งกุญแจความลับนี้ออกไปยังเซิร์ฟเวอร์ภายนอกอื่นใดนอกจาก Google API โดยตรง

Copy to Clipboard: ระบบคลิกคัดลอกโค้ดด่วนด้วยปุ่มเดียวพร้อมสถานะแจ้งเตือน (Toast Notification) ที่สวยงาม

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

HTML5: โครงสร้างเว็บมาตรฐานเชิงความหมาย (Semantic Elements) เช่น <header>, <main>, <section>, <footer>

Tailwind CSS v3 (via Play CDN): สำหรับการตกแต่งสไตล์และออกแบบ Responsive Layout

Vanilla JavaScript (ES6+): ควบคุมตรรกะการประมวลผลและการเรียกเชื่อมต่อ API ของระบบ

Lucide Icons: ชุดไอคอนสไตล์โมเดิร์นและมินิมอล

Gemini API (generativelanguage.googleapis.com): โมเดลปัญญาประดิษฐ์ในการประมวลผลแปลงโค้ด

## 🚀 วิธีเริ่มต้นใช้งาน (Getting Started)

ดาวน์โหลดไฟล์ index.html ไปไว้บนเครื่องคอมพิวเตอร์ของคุณ

ดับเบิลคลิกเปิดไฟล์ index.html ผ่านเว็บเบราว์เซอร์ตัวใดก็ได้ (เช่น Chrome, Edge, Safari, Firefox)

รับสิทธิ์ Gemini API Key ฟรี จาก Google AI Studio

นำ API Key ที่ได้มาวางลงในช่อง "วาง Gemini API Key ที่นี่..." ด้านบนหน้าเว็บ

เลือกทดลองโหลดตัวอย่างผ่านปุ่ม "การ์ดโปรไฟล์" หรือ "ฟอร์มล็อกอิน" หรือวางโค้ด HTML/CSS ของคุณเองในกล่องข้อความฝั่งซ้าย

ตั้งชื่อ Component ที่ต้องการในช่องตั้งชื่อ (เช่น MyComponent)

กดปุ่ม "เริ่มแปลงโค้ด" และรอผลลัพธ์โครงสร้าง TSX + Tailwind แสดงที่ฝั่งขวา

กดปุ่ม "คัดลอกโค้ด" เพื่อนำไปใช้งานต่อในโปรเจกต์ React ของคุณได้ทันที!

## 🔒 นโยบายความปลอดภัยของข้อมูล (Security)

เครื่องมือนี้ทำงานแบบ Client-Side 100% หมายความว่า:

ไม่มีระบบฐานข้อมูลกลางในการจัดเก็บโค้ดของคุณ

API Key ของคุณจะถูกบันทึกไว้ในเครื่องของคุณเองผ่านเทคโนโลยี localStorage ของเบราว์เซอร์เพื่อความสะดวกในการเปิดใช้ครั้งต่อไป

คำขอแปลงโค้ดจะถูกส่งตรงไปยังเซิร์ฟเวอร์ของ Google เท่านั้น ไม่ผ่านระบบตัวแทน (Proxy) ของผู้พัฒนาภายนอก
